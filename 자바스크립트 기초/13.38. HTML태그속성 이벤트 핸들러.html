<이벤트 : 브라우저에서 다양한 이유로 발생하는 사건들> --> 이러한 이벤트에 대처하는 프로그램 : 이벤트 핸들러(주로 함수 하나로 구성됨) 브라우저에서 발생하는 이벤트는 크게 4가지 * Form 이벤트 : HTML 문서의 폼 엘리먼트에 변화가 생겼거나 제출 버튼 등이 눌렸을 때 발생하는 이벤트 * Window 에벤트 : 페이지가 모두 로드되었을 때 발생하는 이벤트로 onload event가 대표적 * MouseEvent : 사용자가 마우스를 조작했을 때 발생하는 이벤트 * KeyboardEvent : 사용자가 키보드를 조작했을 때 발생하는 이벤트 예제:

HTML 태그 속성에 이벤트 핸들러 추가하기

--> click 이벤트는 = 마우스 이벤트(마우스로 클릭했을때 발생) --> keydown 이벤트는 = 키보드 이벤트(키가 눌렸을때 발생) --> change 이벤트는 = 폼 이벤트(form 엘리먼트의 내용이 변경되었을 때 발생)
