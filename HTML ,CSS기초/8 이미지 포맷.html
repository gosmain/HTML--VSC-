<주요 이미지 포맷: GIF,PNG,JPG) * 이미지 포맷을 효율적으로 사용하지 않으면 웹 페이지가 매우 무거워 질 수 있음. <이미지 포맷: 이미지를 압축> (이미지 에디터에서의 저장방법: 포토샴에서 '웹 용으로 저장') 1. JPEG (jpg) = 정지된 사진을 위해 만들어진 손실 압축 방법. * jpg 포맷의 압축은 눈에 잘 띄지는 않지만, 손실되는 방식의 포맷. ------------------------------------------------------ 2. GIF = 움직이는 이미지 (jpg와 달리 비 손실 방식) <하지만 색상에 제한이 있음-권장하지 않는 포맷> * 따라서 로고나 색상이 적은 이미지에서 매우 선명함 * 특허권이 존재해 완전히 자유롭지는 못함 ------------------------------------------------------ 3. PNG(8bit, 24bit) =GIF 포맷의 특허권에 대한 대안 포맷 - PNG 8bit = GIF 포맷과 완전동일(GIF 애니메이션만 제외-따라서 색상역시 제한) * GIF 대체 포맷이며 더욱 압축률이 좋아 이 포맷을 더 권장 * 포토샵에서 Matte라는 옵션을 사용하면 이미지의 퀄리티가 증가 - PNG 24bit = GIF 혹은 PNG8bit보다 색에 대한 제한이 없고 원본의 색을 다 저장할 수 있음. 투명도 또한 더욱 부드러움. * 비손실이지만 이미지 용량이 크게 나옴 (따라서 자연스러운 투명도가 필요하지 않을경우 PNG8bit 혹은 JPG 포맷 권장)
