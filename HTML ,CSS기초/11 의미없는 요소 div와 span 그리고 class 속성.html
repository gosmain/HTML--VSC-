<의미없는 요소 div와 span. 또한 이 요소들을 잘 활용할 수 있는 class 속성, class와 id속성의 차이점, 네이밍> (의미 없는 요소)= div, span (div는 블록 요소, span은 인라인 요소) (특히, div는 레이아웃 잡는 용도로 많이 쓰이기 때문에 중첩됨) --------------------------------------------------- * class 속성 (class와 id는 CSS에서 스타일을 주는데 자주 사용) = div와 span 요소는 보통 class 속성을 같이 사용. = id 속성과 같이 모든 요소에 사용 가능한 속성. = 한 요소에 여러 개의 class 이름을 지정해 줄수 있음. (
) <띄어쓰기로 여러 개의 값 지정 가능> <하나의 네이밍 안에서의 띄어쓰기는 "-"로 구분> -------------------------------------------------- * class와 id의 차이 - id는 페이지에서 딱 한번만 선언 가능(그래야 a 요소로 해당 id 속성에 이동이 가능)(id는 페이지 내 단 한 번만 선언 가능하여, 유일한 이름을 짓는 것이 적합) - class는 여러 번 사용이 가능. (더욱 범용적으로 사용) (class와 id 값의 이름은 무조건 짧게하기 보다는 알기 쉽도록 짓고, 의미에 맞게 넣는 것이 적합)
